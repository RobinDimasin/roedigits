## RoeDigits
Live preview: [https://roedigits.vercel.app/](https://roedigits.vercel.app/)


## Run locally
Install Dependencies
```bash
npm install
# or
yarn install
```

Set up environment variables, see `.env.sample`

Start the server
```bash
npm run dev
# or
yarn dev
```
